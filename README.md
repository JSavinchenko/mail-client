# MailClient (Angular + PrimeNG)

Angular 19.1.6 
Node v22.14.0
npm 10.9.2

## Сборка и запуск

Чтобы запустить локальный сервер:

```bash
ng serve
```
После запуска сервера откройте браузер и перейдите по ссылке `http://localhost:4200/`
